
## Description

- NestJSを初期化した時のサンプルリポジトリです
- [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


### 動作確認した時の各バージョン

| 項目 | 説明 |
|:--- | :---: |
| Node.js | 18.16.0 |
| Nest cli| 10.1.12 |
| Serverless Framework| 3.34.0 |


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
or
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
